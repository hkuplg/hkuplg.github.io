Cubical Type Theory is a powerful mathematical framework, but it is also highly abstract and has an extremely steep learning curve. Trying to learn it without a solid, intuitive understanding makes it feel even harder. During my own learning process, I built a dedicated tool specifically for visualizing the core concepts of Cubical Type Theory. In this talk, I will briefly introduce some of these concepts using this noval visualization tool.
